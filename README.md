# Papocchio

A free-hand draw application for [Sailfish OS](https://sailfishos.org).

## Install

Install via Jolla Store, [Chum](https://github.com/sailfishos-chum/main) or [OpenRepos](https://openrepos.net/content/ilpianista/papocchio).
